# YouTube 채널 동기화 가이드

## 개요
관리자 페이지에서 YouTube 채널의 모든 영상을 자동으로 포트폴리오에 추가할 수 있습니다.

## 사용 방법

### 1. YouTube Data API 키 발급

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. **API 및 서비스** > **라이브러리**로 이동
4. "YouTube Data API v3" 검색 후 **사용 설정** 클릭
5. **API 및 서비스** > **사용자 인증 정보**로 이동
6. **사용자 인증 정보 만들기** > **API 키** 선택
7. 생성된 API 키를 복사 (보안을 위해 제한 설정 권장)

### 2. 채널 동기화 실행

1. 관리자 페이지 로그인
2. **포트폴리오 관리** 메뉴로 이동
3. **YouTube 동기화** 버튼 클릭
4. 다음 정보 입력:
   - **채널 핸들**: `rauvfilm_Cine` (@ 기호 없이)
   - **YouTube Data API 키**: 위에서 발급받은 API 키
   - **카테고리**: 동기화할 영상의 기본 카테고리 선택
5. **동기화 시작** 버튼 클릭

### 3. 동기화 결과

- 채널의 모든 영상이 자동으로 포트폴리오에 추가됩니다
- 이미 존재하는 영상은 건너뜁니다 (중복 방지)
- 영상 제목은 YouTube 제목을 그대로 사용합니다
- 영상 설명은 YouTube 설명을 그대로 사용합니다

## 주의사항

- YouTube Data API는 일일 할당량이 있습니다 (기본 10,000 units)
- 영상 1개 조회 = 약 1 unit 소모
- 할당량 초과 시 다음날까지 대기해야 합니다
- API 키는 안전하게 보관하세요 (환경변수로 관리 권장)

## API 키 보안

프로덕션 환경에서는 API 키를 환경변수로 관리하는 것을 권장합니다:

```env
YOUTUBE_API_KEY=your_api_key_here
```

그리고 코드에서 `process.env.YOUTUBE_API_KEY`를 사용하도록 수정할 수 있습니다.

## 문제 해결

### "채널을 찾을 수 없습니다" 오류
- 채널 핸들이 정확한지 확인하세요
- @ 기호 없이 입력했는지 확인하세요

### "API 키가 유효하지 않습니다" 오류
- API 키가 올바른지 확인하세요
- YouTube Data API v3가 활성화되어 있는지 확인하세요

### "할당량을 초과했습니다" 오류
- 일일 할당량을 초과했습니다
- 다음날까지 대기하거나 할당량 증가를 요청하세요
